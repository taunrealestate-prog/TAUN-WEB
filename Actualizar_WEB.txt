Muy fácil. Como ya tienes:

* VS Code conectado a GitHub ✅
* Vercel conectado al repositorio ✅
* Rama principal `main` ✅

Cada vez que hagas cambios:

### 1. Guarda los archivos

`Ctrl + S`

---

### 2. Abre la terminal en VS Code

```powershell
git status
```

Verás los archivos modificados.

---

### 3. Añade todos los cambios

```powershell
git add .
```

---

### 4. Haz un commit

```powershell
git commit -m "Mejoras estadísticas"
```

(Puedes poner el mensaje que quieras.)

---

### 5. Sube a GitHub

```powershell
git push origin main
```

---

### 6. Espera 20-60 segundos

Vercel detecta automáticamente el cambio y despliega una nueva versión.

Puedes verlo en:

[Panel de Vercel](https://vercel.com/dashboard?utm_source=chatgpt.com)

o directamente recargando:

[taunrealestate.com](https://taunrealestate.com?utm_source=chatgpt.com)

---

### Atajo que usarás el 95% de las veces

```powershell
git add .
git commit -m "Descripción de cambios"
git push origin main
```

Con esos tres comandos tu web queda actualizada online.

Antes de seguir desarrollando, te recomiendo hacer ahora mismo un commit de seguridad:

```powershell
git add .
git commit -m "Sprint responsive y dominio configurado"
git push origin main
```

Así dejamos guardado todo lo que hemos avanzado hoy.
